Advanced AI Prompts Guide

Sophisticated Data Analysis Techniques for Multi-Brand Portfolios

This guide provides advanced prompting techniques for sophisticated data analysis. These prompts are designed for users familiar with the system who want deeper strategic insights from program performance data.

Export Features

For complex analyses, request specific formats: "Present this as an HTML dashboard with exportable charts and include a data table that can be copied to Excel."

The more precise your prompt, the more tailored your results. Include multiple parameters, metrics, and specific data points for optimal analysis.

Operational Optimization

Improve execution efficiency and resource allocation:

Logistics Program Analysis Operations
Analyze all logistics/support programs for [TIME PERIOD]. How do these operational programs impact associated consumer-facing programs? Is there correlation between logistics investment and program success? Identify optimization opportunities. Present in HTML with visualizations and cite specific program data.

Examines relationship between logistics support and program outcomes.

Seasonal Planning Optimization Calendar
Analyze program performance by month across [TIME PERIOD]. Identify seasonal patterns in consumer engagement for each brand. Which months show peak performance for [BRAND A]? For [BRAND B]? Create calendar visualization showing optimal timing for different brands and program types. Include data citations.

Identifies temporal patterns to optimize program scheduling.

Strategic Analysis

Gain deeper strategic insights from your data:

Multi-Brand Event Impact Portfolio
Analyze programs featuring multiple brands versus single-brand programs for [TIME PERIOD]. Do multi-brand activations deliver better overall ROI? Is there evidence of positive spillover effects between brands? Which brand combinations show strongest synergies? Present with exportable visuals and data citations.

Examines whether multi-brand programs create stronger results.

Consumer Journey Mapping Engagement
Map consumer journey across [BRAND] programs from awareness to purchase. Which program sequence creates strongest conversion path? How do sampling events influence subsequent consumption? Create visualization of optimal consumer journey across program types with data citations.

Understands how different program types move consumers through the funnel.

Predictive Performance Model Planning
Based on [TIME PERIOD] data, create predictive model for next quarter performance. What factors most strongly predict program success? Which program types, markets, and brands likely to see strongest performance? Provide specific optimization recommendations with data citations.

Uses historical data for forward-looking projections.

Data Visualization

Create visually compelling presentations of your data:

Executive Dashboard Visual
Create executive dashboard summarizing [TIME PERIOD] performance across entire portfolio. Include high-level metrics, key trends, standout results. Format as HTML with clean professional visualizations. Make charts exportable and include one-page executive summary of key findings and recommendations.

Comprehensive visual overview for executive presentations.

Performance Matrix Multi-Variable
Create matrix visualization comparing program performance across multiple dimensions: brand, market, venue type, program type. Use color coding to highlight high/low performing combinations and identify white space opportunities. Include interactive filtering and data citations.

Multi-dimensional analysis identifying complex patterns.

Creative Analysis

Think about your data in innovative ways:

Scenario Modeling Strategy
Based on [TIME PERIOD] data, model three resource allocation scenarios: 1) Maintain current allocation, 2) Shift 20% to top-performing markets, 3) Shift 20% to top-performing program types. Compare projected outcomes and recommend optimal approach with supporting data.

Evaluates different strategic options through outcome modeling.

Innovation Opportunities Development
Analyze data to identify gaps and opportunities for innovative new program types. Where are untapped consumer segments, underserved markets, or emerging patterns? Based on performance data, what new program formats might deliver strong results? Include data citations to justify recommendations.

Identifies opportunities for program innovation based on data patterns.

Custom Data Extraction

Pull specific datasets for specialized analysis:

Custom Performance Dataset Export
Create comprehensive dataset of all [BRAND] programs from [TIME PERIOD] with fields: program name, market, venue type, dates, attendees, samples, engagements, calculated ROI. Format as HTML table for Excel export. Include summary statistics and highlight top/bottom 10% performers.

Structured dataset for analysis in external tools.

Time Series Analysis Temporal
Extract weekly performance data for all brands across [TIME PERIOD]. Format as time series showing weekly totals for key metrics by brand. Identify seasonal patterns, trends, and anomalies. Present with interactive time series visualizations and data citations.

Chronological dataset identifying temporal patterns.

Advanced Techniques

Master these techniques for sophisticated analysis:

Iterative Analysis

Start high-level, then drill deeper: "Based on your previous analysis, show detailed breakdown of the top 3 performing markets."

Multi-Part Requests

Combine analyses: "Analyze Q2 performance by market, compare ROI across program types, then recommend optimal Q3 allocation."

Hypothesis Testing

Test specific theories: "Test whether programs with higher staff ratios consistently outperform those with lower ratios."

Context Enrichment

Provide background: "Considering our Q2 focus on education, analyze which programs most effectively increased engagement."

Format Specification

Request precise outputs: "Present as HTML dashboard with three sections: Summary, Findings, Recommendations."

Comparative Analysis

Frame for comparison: "Compare engagement metrics for Brand A vs Brand B, highlighting key differences and explanations."

Expert Tip

Combine multiple techniques: "Using Q1-Q2 data, test my hypothesis that [BRAND] performs better in urban markets when paired with education programs, then create an HTML dashboard showing supporting evidence with citations and actionable recommendations for Q3-Q4 planning."
